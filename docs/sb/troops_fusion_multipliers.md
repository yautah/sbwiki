# 融合加成

当你将3个部队合并成一个融合部队时，你会得到一个比之前的部队更强大的部队，但不完全是3倍的强度。

通常，融合的属性加成如下：

- 3.50倍的生命值（+250%）
- 2.05倍的伤害（+105%）
治疗效果通常比伤害增加更多。
